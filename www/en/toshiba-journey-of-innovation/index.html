    • Toshiba: The Journey of Innovation

    Front End Developer

    A website with an experimental mouse gesture navigation system. Interaction is done by drawing signs over the site, a circle, a spiral, a zig-zag.

    Every sign produces a graphical result, every character reacts with a different action. Loading time was used as a tutorial for the navigation system.
